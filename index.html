Inputs Tree Version 1.0 Documenation

Introduction

hello everyone my name is sameh and i'm a web developer introuducing to you the inputs tree application.
this application gives you the ability to make infinite inputs tree and generate the corresponding json data for it and then send this data along with ajax post request
i hope you like it :D

The Mechanism

The basic component in the inputs Tree app is the branch and the branch is a combination of inputs and controllers buttons .


branch sample
each branch name, inputs and controllers get identified in branches in the object that get passed to the inputTree Function that start the app.
and branch name is used to call the branch when you click the controller button
for example in the following code when you click add unit button it calls branch one and append it
mainControllersBtns : [ { text:"Add Unit", control:"branchOne" } ]

branch can be processed as a main parent for other branches if you assigned it's name in the main controllers buttons
mainControllersBtns : [ { text:"Add Unit", control:"branchOne" } ]
OR as a children to other branches if you assigned it's name in any branch sub controllers buttons inside the branch options.
subsControllersBtns:[ { text:"Add Form One", control:"branchOne" }, { text:"Add Form Two", control:"branchTwo" } ]

branches and controllers in this app are used according to the way you idenify it, you can configure each branch inputs and sub controllers as you like it to be.

Why you will like inputs tree?

  • it's easy to install
  • you can make infinite inputs tree using this app.
  • you can generate full json data corresponding to the tree you created.
  • you can send the generated json data along with ajax post request.

How To Use?

  • 1- Html markup
    <div id="inputs_tree_action_btns"></div>
    <div id="inputs_tree_main_controllers"></div>
    <div id="inputs_tree_holder"></div>

  • 2- Call the app files
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.4/jquery.min.js"> </script>
    <script src="js/inputsTree.js"></script>
    <script src="js/inputsTreeOptions.js"></script>

  • 3- Configure "js/inputsTreeOptions.js" according to your needs
    in this documentaion you will find the options table that explain each option in this file

  • 4- Call inputsTree function and pass the component object in order to start the app
    <script>window.inputsTree( inputsTreeComponenets );</script>

General notes about The Options Table

  • 1- "options" and "DataSourceUrl" in the following options table are two diffrent way for producing select options data which means you have to assign one option only
  • 2- data that sent with ajax post request being sent in array with key 'it_data'.
    so to handle this data in the server side file you have to call the array with key data.
    for example in php in order to print the data
    print_r( $_POST['it_data'] );
